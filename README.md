# The Gambia-air-pollution-pneumococcal-colonisation

Longitudinal personal environmental exposure and *Streptococcus pneumoniae*
colonisation in a cohort of children 5-14 years old in rural Gambia.

**Status:** Manuscript in preparation.

**Code:** The analysis code will be published here once the manuscript is
submitted.

## Summary

This project investigates associations between personal environmental
exposures including PM2.5, carbon monoxide, temperature, and relative
humidity, measured using PATS+ personal monitors and *S. pneumoniae*
colonisation in a longitudinal cohort of older children (5–14 years) in rural
Gambia. Colonisation status was determined by microbiological culture at each
of eight study visits, and associations were examined using mixed-effects
models, including a within-child/between-child exposure decomposition
approach.
